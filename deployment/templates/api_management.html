API Documentation

Use your API key to access WiseNews programmatically:

Authentication
curl -H "X-API-Key: YOUR_API_KEY" \ https://wisenews.app/api/articles
Available Endpoints
  • GET /api/articles
  • Retrieve news articles
  • GET /api/search
  • Search articles
  • GET /api/categories
  • Get article categories
Rate Limits
  • 1000 requests/day
  • 60 requests/hour
  • 10 requests/minute

Security Tips
  • Keep your API keys secure and private
  • Use environment variables in production
  • Rotate keys regularly
  • Monitor usage for unusual activity
  • Never commit keys to version control
